# bech32

A BIP173/BIP350 compatible Bech32/Bech32m encoding/decoding library for Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

  ```yaml
  dependencies:
    bech32:
      github: wout/bech32
  ```

2. Run `shards install`
